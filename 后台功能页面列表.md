# 后台功能页面列表

## 管理功能页面
1. 用户管理
   - 用户信息查看、编辑
   - 权限分配
   - 登录日志查看
2. 会场管理
   - 会场配置（名称、地点、状态）
   - 会期时间配置
   - 主席团成员管理
   - 议程管理
3. 代表管理
   - 代表信息导入、编辑
   - 国家分配
   - 点名状态管理
4.  日志管理
    - 操作日志查看
    - 审计记录
    - 

## 主席团功能页面
1. 文件审批
   - 文件内容查看
   - 文件审批与反馈
2. 文件管理
   - 文件审批、发布
   - 文件分类（立场文件、工作文件等）
   - 文件可见性设置
3. 危机管理
   - 危机发布
   - 危机反馈查看
4. 时间轴管理
   - 时间轴事件记录
   - 时间流速设置
5. 消息管理
   - 消息发送
   - 消息记录查看
   -   
## 选手功能页面
1. 个人面板
   - 个人信息查看与修改
   - 当前会场状态查看
2. 文件相关
   - 文件上传
   - 文件查看
3. 消息中心
   - 接收主席团消息
   - 私聊与群聊
4. 危机响应
   - 危机内容查看
   - 危机反馈提交
